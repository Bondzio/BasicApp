UDP

(Informatik, Technik)

Synonym:

User Datagram Protocol

Definition:

Das User Datagram Protocol, kurz UDP, ist ein minimales, verbindungsloses Netzwerkprotokoll, das zur Transportschicht der Internetprotokollfamilie gehört. UDP ermöglicht Anwendungen den Versand von Datagrammenin IP-basierten Rechnernetzen.
