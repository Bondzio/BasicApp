Eingabe

\

(Informatik, Technik)

\
\
\
\
\

Synonym:

\

Input

\
\
\

Definition:

\

Die Eingabe eines Computerprogramms ist das, was es zu seiner Ausführung benötigt. Der Begriff Eingabe bezeichnet dabei sowohl die Daten als auch den Vorgang, mit dem diese an das Programm übergeben werden. Der Begriff bezeichnet auch innerhalb eines Programms die Übergabe von Daten an dessen Teilstrukturen wie Funktionen, Prozeduren  oder Operationen über Parameter, die dann Eingabeparameter genannt werden.

\
\
\

Beispiel:

\

Die Eingabe erfolgt beispielsweise über Eingabegeräte wie Maus oder Tastatur oder es werden Daten aus einer Datei eingelesen oder über ein Computernetz von einem anderen Computer übertragen.
