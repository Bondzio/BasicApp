<h1>Parameter</h1>

<p>(Informatik, Mathematik)</p>

</div>

<div class="container">

<div class="row">

<div class="col-sm-4">

<h3>Synonym:</h3>

<p>&Uuml;bergabewert (Informatik), Formvariable (Mathematik)</p>

</div>

<div class="col-sm-4">

<h3>Definition:</h3>

<p>Je nach Kontext kann ein Parameter eine unterschiedliche Bedeutung haben.</p>

</div>

<div class="col-sm-4">

<h3>Verwendung in der Informatik:</h3>

<p>Parameter sind in der&nbsp;Informatik&nbsp;Variablen, durch die ein&nbsp;Computerprogramm&nbsp;(oft ein&nbsp;Unterprogramm) auf die Verarbeitung bestimmter Werte &bdquo;eingestellt&ldquo; werden kann. Parameter sind also programmextern gesetzte&nbsp;Einflussfaktoren; sie werden insbesondere beim Aufruf von Unterprogrammen verwendet, um diesen &sbquo;mitzuteilen&lsquo;, welche Daten/Werte sie verarbeiten sollen und ggf. wie.</p>

</div>

<div class="col-sm-4">

<h3>Verwendung in der Mathematik:</h3>

<p>Als Parameter wird in der Mathematik eine&nbsp;Variable&nbsp;bezeichnet, die gemeinsam mit anderen Variablen auftritt, aber von anderer Qualit&auml;t ist. Man spricht auch davon, dass ein Parameter beliebig, aber fest ist. Er unterscheidet sich damit von einer&nbsp;Konstanten&nbsp;dadurch, dass der Parameter nur f&uuml;r einen gerade betrachteten Fall konstant ist, f&uuml;r den n&auml;chsten Fall aber variiert werden kann.</p>

  </div>

  <div class="col-sm-4">

<h3>Beispiel:</h3>

<p>In der Informatik kann eine Operation zum Beispiel read, write, commit oder abort sein. </p>

</div>

</div>
