Bedingung

(Informatik)

Definition:

Eine Bedingung ist ein logischer Term, welcher zur Verwicklung von etwas anderem (im Informatik Kontext meist eine Anweisung) als Vorraussetzung gegeben sei muss.

Eine Bedingung kann als true oder false ausgewertet werden und dementsprechend wird dann die nächste Anweisung ausgeführt.
